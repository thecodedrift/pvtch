## Translating Chat

When translating messages from chat, you'll send the entire message, along with the sender's username. If the language is different that your **target language**, you'll get back a translated version of the text for use in your bot's software.

```
[nl] This is translated text from Dutch (nl) to English (en)
```

  <TabItem label="Firebot">
    <h3>Create a new Event</h3>
    In the Firebot app, go to the **Events** section and create a new event associated with `Chat Message (Twitch)`

    <img src={firebotCreateEvent.src} alt="Creating a new Firebot Event" class="my-4 border rounded-lg shadow-md" />

    <h3>The Three Event Effects</h3>
    Your Firebot Event will have three effects:

    1. HTTP Request (send the message to Lingo)
    2. Conditional Effects (chat only if there is a translation)
    3. Conditional Effect => Chat (send translation to your channel)

    <h3>1. HTTP Request Effect</h3>
    In the HTTP Request effect, set it up as follows:

    **URL**: Use the following URL, unique to your Lingo + Firebot configuration:
    <TranslateAllURL client:only replacements={[["SENDINGUSER", "$user"], ["MESSAGEHERE", "$encodeForUrl[$chatMessage]"]]} />

    **Method**: GET

    <img src={firebotHttpRequest.src} alt="Firebot HTTP Request Effect Setup" class="my-4 border rounded-lg shadow-md" />

    <h3>2. Conditional Effects</h3>
    Add a Conditional Effect to check if the response from the HTTP Request got any data back.

    **Condition (all)**: `$effectOutput[httpResponse]` is not ` ` (blank)

    **Effect**: Chat

    <img src={firebotConditional.src} alt="Firebot Conditional Effect Setup" class="my-4 border rounded-lg shadow-md" />

    <h3>3. Chat Effect (in conditional)</h3>
    In the Chat effect, set it up as follows so it'll reply to the original poster with their translation

    **Chat as**: Bot

    **Message to Send**: `ImTyping $effectOutput[httpResponse]`

    **Send as reply**: ✅

    <img src={firebotChat.src} alt="Firebot Chat Effect Setup" class="my-4 border rounded-lg shadow-md" />

  <TabItem label="MixItUp">
    <h3>Create a new Chat Message Received Event</h3>
    
    The Chat Message Received event will trigger whenever a message is sent in your chat. You'll be encoding the message, sending it to PVTCH, and then checking if your bot needs to reply. You can find the event under `Events` => `Chat` (not Twitch).

    <img src={mixitupChatReceived.src} alt="Creating a Chat Received Event in MixItUp" class="my-4 border rounded-lg shadow-md" />

    <h3>The Three Event Actions</h3>
    Your MixItUp Event will have three actions:

    1. Special Identifer Action - to prepare the message for PVTCH
    2. Web Request Action - send the message to PVTCH
    3. Conditional Effect Action + Chat Action - send translation to your channel

    <h3>1. Special Identifier Action</h3>

    **Special Identifier Name**: `encodedMessage`

    **Value**: `uriescape($message)`

    Make Globally Usable: ❌ \
    Is Math Expression: ❌ \
    Replace Special Identifiers In Functions: ❌

    <img src={mixitupSpecialIdentifier.src} alt="Setting Up the MixItUp Special Identifier" class="my-4 border rounded-lg shadow-md" />

    <h3>2. Web Request Action</h3>

    **Web Request URL**: Use the following URL, unique to your Lingo + Firebot configuration:
    <TranslateAllURL client:only replacements={[["SENDINGUSER", "$user"], ["MESSAGEHERE", "$encodedMessage"]]} />

    **Response Processing Type**: `Plain Text`

    <img src={mixitupWebRequest.src} alt="The MixItUp Web Request Action" class="my-4 border rounded-lg shadow-md" />

    <h3>3. Conditional and Chat Actions</h3>
