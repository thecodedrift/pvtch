**PVTCH** was built to make great streaming tools accessible to everyone.

The project's goals:

1. Easy to Use - No coding required, just bot and OBS setup
2. Minimal Permissions - Only your Twitch ID is needed (so we can talk to offenders)
3. At Cost - Try to do this for **FREE** and give everyone the ability to self-host
